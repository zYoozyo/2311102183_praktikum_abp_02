## API Endpoints

| Method   | Endpoint        | Fungsi           |
|----------|-----------------|------------------|
| GET      | `/api/data`     | Ambil semua data |
| GET      | `/api/data/:id` | Ambil satu data  |
| POST     | `/api/data`     | Tambah data baru |
| PUT      | `/api/data/:id` | Update data      |
| DELETE   | `/api/data/:id` | Hapus data       |
